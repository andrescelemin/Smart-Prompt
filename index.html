Desarrollo Iterativo

Como todo el machine learning, el desarrollo de prompts efectivos es un proceso iterativo. Es muy improbable encontrar el prompt adecuado en un primer intento. Lo normal es tener que dedicar mucho tiempo a generar prompts específicos para resolver tareas concretas.

Framework de la Implementación de un Prompt

  • Análisis de errores y resultados
  • Idea/problema a resolver
  • Análisis e implementación del prompt
  • Resultados experimentales

Seguridad y Evitación de Problemas

Evitamos problemas de seguridad como los prompt injections.

Prompt Injection se refiere al proceso de manipular la salida de un modelo de lenguaje mediante la inserción de texto no autorizado en sus instrucciones. Esta técnica permite a un atacante hacer que el modelo genere cualquier contenido de su elección, a menudo con intenciones maliciosas. Nosotros implementamos medidas para evitar estos problemas y asegurar la integridad del sistema.
